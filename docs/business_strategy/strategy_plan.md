## Part 1: 戦略概要

### ポジショニング

```
【競合】                    【本アプリ】
汎用介護ソフト              ニッチ特化型
├─ 高機能・高価格           ├─ シンプル・低価格
├─ 請求機能中心             ├─ 監査対応・記録中心
└─ 全施設種別対応           └─ 障がい者GH × 訪問看護特化
```

### 差別化の3軸

| 軸 | 内容 |
|----|------|
| **価格** | 月額3,000〜5,000円（競合の1/3〜1/5） |
| **シンプル** | 必要機能のみ、直感的UI |
| **ニッチ特化** | 障がい者GH + 訪問看護の実務に完全対応 |
