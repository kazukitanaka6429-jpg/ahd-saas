# 業務日誌の個別保存機能

## 要件
- 一人ずつバリデーション・保存を行う
- 日中活動：GH、日中活動、その他福祉サービスが全て未入力の場合、3つ全てをエラー表示
- 夜間：GH泊だけでなく、すべての夜間項目（GH泊、救急搬送、入院、外泊）をエラー表示
- 入力不備のある人だけが保存されないようにする
