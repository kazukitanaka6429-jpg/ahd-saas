## 4. 事業価値算定

### 4.1 算定方法

3つの独立した手法で算定し、加重平均で最終値を導出。

### 4.2 手法① コスト積上法（再構築コスト）

> 「同等のシステムをゼロから開発した場合のコスト」

| 項目 | 数量 | 単価 | 金額 |
|---|---|---|---|
| フロントエンド開発 | 120人日 | ¥50,000/日 | ¥6,000,000 |
| バックエンド/DB設計 | 60人日 | ¥60,000/日 | ¥3,600,000 |
| テスト・QA | 20人日 | ¥40,000/日 | ¥800,000 |
| DevOps・インフラ | 15人日 | ¥50,000/日 | ¥750,000 |
| 業務要件定義・設計 | 30人日 | ¥70,000/日 | ¥2,100,000 |
| **再構築コスト合計** | | | **¥13,250,000** |
| 技術的負債割引 (-40%) | | | -¥5,300,000 |
| **調整後コスト** | | | **¥7,950,000** |

### 4.3 手法② 収益還元法（DCF簡易版）

> 「将来キャッシュフローの現在価値」

**前提条件**:
- 現在の顧客: 1事業所（自社利用）
- 月額想定: ¥30,000〜50,000/事業所
- 1年後目標: 5事業所、3年後: 30事業所
- 成長率仮定: 年200%（初期フェーズ）
- 割引率: 30%（アーリーステージリスク）

| 年 | 事業所数 | 月額MRR | 年間ARR | 割引後 |
|---|---|---|---|---|
| Year 1 | 3 | ¥120,000 | ¥1,440,000 | ¥1,108,000 |
| Year 2 | 10 | ¥400,000 | ¥4,800,000 | ¥2,840,000 |
| Year 3 | 30 | ¥1,200,000 | ¥14,400,000 | ¥6,559,000 |
| **NPV合計** | | | | **¥10,507,000** |
| 実現確率割引 (-60%) | | | | -¥6,304,000 |
| **調整後DCF** | | | | **¥4,203,000** |

> [!CAUTION]
> 顧客ゼロ（自社利用のみ）のため、収益実績なし。実現確率を厳しく割り引いています。

### 4.4 手法③ 市場比較法（マルチプル）

> 「類似SaaS企業の評価倍率を適用」

| 比較対象 | ARRマルチプル | 備考 |
|---|---|---|
| 国内SaaS中央値 | 8〜12x ARR | 上場済み |
| アーリーステージSaaS | 3〜5x ARR | プレシード |
| 介護・福祉SaaS | 5〜8x ARR | ドメイン特化 |

- 現在ARR: ¥0（外部顧客なし）
- 潜在ARR（自社1事業所仮定）: ¥480,000
- マルチプル 5x 適用: **¥2,400,000**

### 4.5 最終算定

| 手法 | 算定額 | 重み | 加重額 |

| DCF法 | ¥4,203,000 | 30% | ¥1,261,000 |

## 5. リスク要因（減価要因）

| リスク | 影響度 | 対策 |
|---|---|---|
| 外部顧客ゼロ | **致命的** | PMF検証が最優先。1社でも外部導入で価値2〜3倍 |
| テスト不足 | **高** | 監査ロジック・請求計算の自動テスト必須 |
| RLSの複雑さ | **高** | Admin Client依存を減らし、RLSポリシーをリファクタリング |
| チーム規模 | **高** | 1人開発は継続性リスク。ドキュメント・テストで軽減 |
| 法規制対応 | **中** | 介護保険法・障害者総合支援法の改正への追従が必要 |
| Supabase依存 | **中** | ベンダーロックイン。PostgreSQL準拠のため移行は可能 |

## 7. 結論

> **本プロダクトの現時点での推定事業価値は約 ¥5,700,000（570万円）です。**

**ポジティブ要因**:
- 特定業界に深く刺さるドメイン知識が組み込まれている
- 人員配置監査・医療連携Ⅳロジック等は競合優位性となりうる
- モダンな技術スタック（Next.js 16 / React 19）で拡張性が高い
- マルチテナント設計が既に組み込まれている

**ネガティブ要因**:
- 外部顧客ゼロ（PMF未検証）
- テストカバレッジが極めて低い
- RLSポリシーの技術的負債が蓄積
- 1人開発体制の継続性リスク

> [!IMPORTANT]
> **外部顧客を1社でも獲得すれば、価値は ¥10,000,000〜¥15,000,000 に跳ね上がる可能性があります。** 現時点の価値の大部分は「再構築コスト」に基づいており、PMF検証後に収益ベースの評価に移行できます。
