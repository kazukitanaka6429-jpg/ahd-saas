#### Sentry.io とは？

**Sentry**は、アプリケーションのエラー監視・パフォーマンス監視を行うクラウドサービスです。

| 機能 | 説明 |
|------|------|
| **エラー追跡** | 本番環境で発生したエラーをリアルタイムで検知・通知 |
| **スタックトレース** | エラーの発生箇所・原因を詳細に表示 |
| **ユーザーセッション** | どのユーザーがエラーに遭遇したかを追跡 |
| **パフォーマンス監視** | ページ読み込み速度やAPIレスポンス時間を計測 |
| **アラート通知** | Slack、メール等でエラー発生を即時通知 |

---

#### なぜ必要？

1. **本番環境のエラーが見えない問題**
   - ユーザーが遭遇したエラーを開発者が把握できない
   - 「動かない」という報告を受けても原因がわからない

2. **問題の早期発見**
   - デプロイ後のバグを即座に検知
   - 影響を受けたユーザー数を把握

3. **デバッグ効率化**
   - エラー発生時の状況（ブラウザ、OS、ログイン状態等）を自動記録
   - 再現手順を推測しやすい

4. **品質向上**
   - エラー傾向の分析
   - 優先度付けて修正

#### 設定手順（契約後）

1. [sentry.io](https://sentry.io) でアカウント作成
2. 新規プロジェクト作成（Platform: Next.js）
3. 以下の環境変数を `.env.local` に追加：

```env
NEXT_PUBLIC_SENTRY_DSN=https://xxxx@xxx.ingest.sentry.io/xxxx
SENTRY_ORG=your-org-name
SENTRY_PROJECT=yorisol
SENTRY_AUTH_TOKEN=sntrys_xxxxx
```

4. Vercelにも同じ環境変数を設定
5. 動作確認
