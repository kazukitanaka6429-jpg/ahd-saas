# 経営ダッシュボード (HQ Dashboard) アーカイブ

このディレクトリには、将来的に「経営ダッシュボード」機能が再度必要になった場合に備えて、作成途中のコード一式を保存しています。

## 保存内容
- `app/` : 画面コンポーネントおよびルーティング (`page.tsx`, `dashboard-client.tsx`)
- `components/` : ダッシュボード専用コンポーネント (KPIグラフ、アラートリスト、出力ボタンなど)
- `actions/` : バックエンドと通信する Server Actions (`dashboard.ts`, `report.ts` など)

## 備考
ユーザーから「既存の機能・デザインのみの改修でよい」との指定があったため、意図せず混入していた本機能をメインツリーから切り離してここにバックアップしています。
